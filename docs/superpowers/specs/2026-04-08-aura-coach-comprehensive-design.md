# Aura Coach AI — Comprehensive Design Specification

**Date:** 2026-04-08  
**Version:** 1.1  
**Status:** Approved for Implementation (v1.1 — added Notification feature)  
**Stack:** Flutter 3.x (Dart ≥3.2) + Firebase + Gemini SDK + Provider + GoRouter v17+

## 1. Architecture Overview

### 1.1 Technology Stack

| Layer | Technology | Purpose |
|-------|-----------|---------|
| **Framework** | Flutter 3.x (Dart ≥3.2) | Cross-platform mobile (iOS/Android) |
| **State Mgmt** | Provider (ChangeNotifier) | Feature-scoped reactive state |
| **Navigation** | GoRouter v17+ | Type-safe routing + deep linking |
| **Auth** | Firebase Auth (Google/Apple/Anonymous) | Native SSO, no WebView |
| **Database** | Cloud Firestore | NoSQL document store |
| **AI Engine** | Google Generative AI SDK (`google_generative_ai`) | Direct Gemini API calls (no Cloud Functions) |
| **Local Cache** | SharedPreferences | Offline-first conversation cache |
| **Images** | Cloudinary CDN (`cached_network_image`) | Clay 3D icons + avatars + mode cards |
| **TTS** | Google Cloud Text-to-Speech API | Audio pronunciation |
| **UI Icons** | `flutter_lucide` (52 vector icons) | Action + state icons |
| **Topic Icons** | `fluentui_emoji_icon` (16 static 3D emojis) | Topic selection emojis |
| **Launcher Icons** | `flutter_launcher_icons` | Custom app icon generation |

### 1.3 Architecture Principles

**One Border Per Component Rule:**
- ClayCard owns the border. NO child containers should have their own borders.
- Selection state changes the card's border color via `isSelected` prop.
- Use `selectedBorderColor` to define the accent on selection.

**Direct Gemini Calls:**
- All AI calls go through `GeminiDatasource` directly to the Gemini API.
- No Cloud Functions proxy. Client-side calls reduce latency and simplify deployment.
- API key stored in `.env` and loaded at runtime.

**Dual-Write Persistence:**
- All writes hit SharedPreferences first (immediate UI response).
- Then write to Firestore asynchronously (background sync).
- Failed Firestore writes are queued for retry.

**Feature-Scoped Providers:**
- Each feature (scenario, story, translator, vocab) has its own Provider.
- Providers manage UI state (loading, error, messages, assessment results).
- Repositories injected via Provider parameters, not global singletons.

## 2. Core Layer — Design Tokens & Configuration

### 2.1 Colors (ALL with Hex Codes)

**Surface Colors:**
- `cream`: #FFF8F0 — Main background
- `clayWhite`: #FEFCF9 — Card backgrounds
- `clayBeige`: #F5EDE3 — Input fields, inactive states
- `clayBorder`: #E8DFD3 — Default border color
- `clayShadow`: #D4C9BB — Shadow color base

**Text Colors:**
- `warmDark`: #2D3047 — Primary text, headings
- `warmMuted`: #6B6D7B — Secondary text, labels
- `warmLight`: #9B9DAB — Tertiary text, disabled
- `white`: #FFFFFF — Inverted text on dark backgrounds

**Accent Colors:**
- `tealClay`: #7ECEC5 — Primary action, active states
- `purpleClay`: #A78BCA — Secondary accent
- `goldClay`: #E8C77B — Tertiary accent, highlights

**Semantic Colors:**
- `success`: #7BC6A0 — Success states (green)
- `warning`: #E8C77B — Warning states (gold)
- `error`: #D98A8A — Error states (red)
- `info`: #7ECEC5 — Info states (teal)

**Tone Colors (for Tone Translator):**
- `formalTone`: #6366F1 — Formal tone (indigo)
- `neutralTone`: #7BC6A0 — Neutral tone (teal)
- `friendlyTone`: #E8C77B — Friendly tone (gold)
- `casualTone`: #D98A8A — Casual tone (rose)

**SM2Data (Spaced Repetition Algorithm):**
```dart
class SM2Data {
  int interval; // Days until next review
  double easeFactor; // Difficulty multiplier
  DateTime nextReviewAt;

  SM2Data({
    this.interval = 1,
    this.easeFactor = 2.5,
    required this.nextReviewAt,
  });

  void updateAfterReview(bool wasCorrect) {
    if (wasCorrect) {
      easeFactor = (easeFactor - 0.2 + 0.1).clamp(1.3, 10.0);
      interval = (interval * easeFactor).round();
    } else {
      interval = 1;
      easeFactor = ((easeFactor - 0.2).clamp(1.3, 10.0));
    }
    nextReviewAt = DateTime.now().add(Duration(days: interval));
  }
}
```

```dart
  "pronunciation": "/..."
}
''';

    final response = await _model.generateContent([Content.text(prompt)]);
    return _parseJson(response.text);
  }

  /// Function 8: Generate mind map for a word
  Future<MindMap> generateMindMap(String word) async {
    final branches = await generateMindMapBranches(word, 5);
    return MindMap(
      id: Uuid().v4(),
      userId: '', // Set by caller
      rootWord: word,
      nodes: [
        MindMapNode(id: '0', text: word, level: 0, childIds: List.generate(branches.length, (i) => '$i')),
        ...branches.asMap().entries.map((e) => MindMapNode(
          id: '${e.key}',
          text: e.value,
          level: 1,
          childIds: [],
        )),
      ],
      createdAt: DateTime.now(),
      updatedAt: DateTime.now(),
    );
  }

  /// Function 9: Generate mind map branches for a word
  Future<List<String>> generateMindMapBranches(String word, int maxBranches) async {
    final prompt = '''
Create a mind map for the word: "$word"

Generate $maxBranches related words/concepts that branch from "$word":
- Related meanings
- Common collocations
- Semantic fields
- Word family members

Return JSON:
{
  "branches": ["word1", "word2", "word3", ...]
}
''';

    final response = await _model.generateContent([Content.text(prompt)]);
    final json = _parseJson(response.text);
    return List<String>.from(json['branches'] ?? []);
  }

  // ═════════════════════════════════════════════════════════════════
  // ASSESSMENT Functions (Reused across modes)
  // ═════════════════════════════════════════════════════════════════

  /// Function 10-14: Various specific assessments
  /// (generateScenarioAssessment, generateStoryAssessment, etc.)
  /// These are thin wrappers around evaluateUserResponse with context-specific prompts.

  // Helper: Parse JSON from model response
  Map<String, dynamic> _parseJson(String? text) {
    if (text == null) return {};
    try {
      // Extract JSON from markdown code blocks if present
      final jsonMatch = RegExp(r'\{[\s\S]*\}').firstMatch(text);
      final jsonStr = jsonMatch?.group(0) ?? text;
      return jsonDecode(jsonStr);
    } catch (e) {
      return {};
    }
  }
```

#### Architecture

**ScenarioProvider:**
```dart
class ScenarioProvider extends ChangeNotifier {
  final AiRepository _aiRepository;
  final PersistenceRepository _persistenceRepository;
  final UsageRepository _usageRepository;

  String? _currentConversationId;
  Conversation? _currentConversation;
  AssessmentResult? _latestAssessment;
  bool _isLoading = false;
  String? _error;
  List<Lesson> _availableLessons = [];

  // Getters...

  Future<void> initiateLessonSelection(String topic) async {
    _isLoading = true;
    _error = null;
    notifyListeners();
    try {
      // Fetch pre-defined lessons for topic from Firestore
      _availableLessons = await _persistenceRepository.getLessonsForTopic(topic);
    } catch (e) {
      _error = 'Failed to load lessons';
    }
    _isLoading = false;
    notifyListeners();
  }

  Future<void> startScenario(String lessonId, String topic) async {
    _isLoading = true;
    _error = null;
    notifyListeners();
    try {
      // 1. Create new conversation
      _currentConversationId = Uuid().v4();
      
      // 2. Generate AI lesson + opening line
      final lessonContext = await _aiRepository.generateScenarioLesson(
        topic,
        userProfile.proficiencyLevel,
      );
      
      // 3. Create first AI message
      final aiMessage = Message(
        id: Uuid().v4(),
        speaker: 'ai',
        text: lessonContext,
        timestamp: DateTime.now(),
      );
      
      // 4. Initialize conversation
      _currentConversation = Conversation(
        id: _currentConversationId!,
        userId: authProvider.currentUser!.uid,
        mode: 'roleplay',
        topic: topic,
        messages: [aiMessage],
        status: 'in-progress',
        createdAt: DateTime.now(),
        updatedAt: DateTime.now(),
      );
      
      // 5. Save locally (immediate) + to Firestore (background)
      await _persistenceRepository.saveConversation(_currentConversation!);
      
      // 6. Increment daily usage
      await _usageRepository.incrementUsage(authProvider.currentUser!.uid, 'roleplay');
    } catch (e) {
      _error = 'Failed to start scenario';
    }
    _isLoading = false;
    notifyListeners();
  }

  Future<void> submitUserResponse(String userText) async {
    if (_currentConversation == null) return;
    _isLoading = true;
    _error = null;
    notifyListeners();
    try {
      // 1. Add user message
      final userMessage = Message(
        id: Uuid().v4(),
        speaker: 'user',
        text: userText,
        timestamp: DateTime.now(),
      );
      _currentConversation!.messages.add(userMessage);
      
      // 2. Evaluate user response (AI assessment)
      final assessment = await _aiRepository.evaluateResponse(
        userText,
        _currentConversation!.messages.map((m) => '${m.speaker}: ${m.text}').join('\n'),
        userProfile.proficiencyLevel,
      );
      userMessage.assessment = assessment;
      _latestAssessment = assessment;
      
      // 3. Generate AI follow-up (next scenario step or feedback)
      final aiFollowUp = await _aiRepository.generateScenarioLesson(
        _currentConversation!.topic,
        userProfile.proficiencyLevel,
      );
      final aiMessage = Message(
        id: Uuid().v4(),
        speaker: 'ai',
        text: aiFollowUp,
        timestamp: DateTime.now(),
      );
      _currentConversation!.messages.add(aiMessage);
      
      // 4. Save updated conversation
      await _persistenceRepository.saveConversation(_currentConversation!);
      
      // 5. Update total score
      _currentConversation!.totalScore = _calculateAverageScore();
    } catch (e) {
      _error = 'Failed to process response';
    }
    _isLoading = false;
    notifyListeners();
  }

  Future<void> endConversation() async {
    if (_currentConversation == null) return;
    _isLoading = true;
    try {
      _currentConversation!.status = 'completed';
      _currentConversation!.updatedAt = DateTime.now();
      await _persistenceRepository.saveConversation(_currentConversation!);
    } catch (e) {
      _error = 'Failed to save conversation';
    }
    _isLoading = false;
    notifyListeners();
  }

  // Continue same conversation from history
  Future<void> resumeConversation(String conversationId) async {
    _isLoading = true;
    try {
      _currentConversation = await _persistenceRepository.getConversation(conversationId);
      _currentConversationId = conversationId;
    } catch (e) {
      _error = 'Failed to load conversation';
    }
    _isLoading = false;
    notifyListeners();
  }

  int _calculateAverageScore() {
    if (_currentConversation?.messages.isEmpty ?? true) return 0;
    final scores = _currentConversation!.messages
        .where((m) => m.assessment != null)
        .map((m) => m.assessment!.overallScore)
        .toList();
    return (scores.isEmpty) ? 0 : (scores.reduce((a, b) => a + b) / scores.length).toInt();
  }
}
```

```dart
class TranslatorProvider extends ChangeNotifier {
  final AiRepository _aiRepository;
  final TtsRepository _ttsRepository;
  final PersistenceRepository _persistenceRepository;

  String _inputText = '';
  Map<String, String> _toneVariations = {};
  Map<String, String> _audioUrls = {}; // Cached TTS URLs
  bool _isLoading = false;
  String? _error;

  // Getters...

  Future<void> generateToneVariations(String text) async {
    _inputText = text;
    _isLoading = true;
    _error = null;
    notifyListeners();
    try {
      _toneVariations = await _aiRepository.generateAllToneVariations(text);
      
      // Pre-generate TTS for each tone
      for (final tone in _toneVariations.keys) {
        final audioUrl = await _ttsRepository.generateSpeech(_toneVariations[tone]!, 'en-US');
        _audioUrls[tone] = audioUrl;
      }
    } catch (e) {
      _error = 'Failed to generate tones';
    }
    _isLoading = false;
    notifyListeners();
  }

  Future<void> saveToneToDictionary(String tone) async {
    try {
      final item = SavedItem(
        id: Uuid().v4(),
        userId: authProvider.currentUser!.uid,
        type: 'tone-variation',
        originalText: _inputText,
        savedText: _toneVariations[tone]!,
        audioUrl: _audioUrls[tone],
        dictionaryData: null,
        sm2: SM2Data(nextReviewAt: DateTime.now()),
        createdAt: DateTime.now(),
        updatedAt: DateTime.now(),
      );
      await _persistenceRepository.saveItem(item);
    } catch (e) {
      _error = 'Failed to save';
    }
    notifyListeners();
  }

  Future<void> copyToClipboard(String text) async {
    await Clipboard.setData(ClipboardData(text: text));
  }
}
```

```dart
    }
```

```dart
class LibraryProvider extends ChangeNotifier {
  final PersistenceRepository _persistenceRepository;

  List<SavedItem> _items = [];

  String _selectedCategory = 'All';
  String _selectedTag = 'All';
```

### Phase 8: Notifications (Learning Reminders)

**Screens:** NotificationCenterScreen, NotificationPermissionSheet (onboarding)  
**Routes:** `/notifications`  
**Duration:** Week 7-8

#### 8.1 Overview

Aura Coach uses Firebase Cloud Messaging (FCM) to deliver four types of learning reminders that keep users engaged and maintain their learning streaks. All notifications are local-first where possible (scheduled on-device) with FCM as the fallback for server-triggered events.

**Notification Types:**

| Type | Trigger | Priority | Default Time |
|------|---------|----------|-------------|
| **Daily Learning Reminder** | User hasn't opened app today | Normal | 09:00 (user-configurable) |
| **Streak Warning** | User has active streak ≥3 days and hasn't practiced today by evening | High | 20:00 (user-configurable) |
| **Spaced Repetition Review** | SM-2 algorithm determines flashcards are due for review | Normal | 10:00 (batched daily) |
| **Quota Reset** | Daily quota resets at midnight (free tier users only) | Low | 00:05 local time |

#### 8.2 Architecture

**NotificationProvider:**
```dart
class NotificationProvider extends ChangeNotifier {
  final NotificationRepository _notificationRepository;
  final NotificationScheduler _scheduler;

  NotificationPreferences _preferences = NotificationPreferences.defaults();
  List<NotificationItem> _notifications = [];
  int _unreadCount = 0;
  bool _isLoading = false;

  NotificationPreferences get preferences => _preferences;
  List<NotificationItem> get notifications => _notifications;
  int get unreadCount => _unreadCount;
  bool get isLoading => _isLoading;

  Future<void> initialize(String userId) async {
    _isLoading = true;
    notifyListeners();
    try {
      _preferences = await _notificationRepository.getPreferences(userId);
      _notifications = await _notificationRepository.getNotifications(userId, limit: 50);
      _unreadCount = _notifications.where((n) => !n.isRead).length;
      await _scheduler.scheduleAll(_preferences);
    } finally {
      _isLoading = false;
      notifyListeners();
    }
  }

  Future<void> updatePreferences(NotificationPreferences updated) async {
    _preferences = updated;
    await _notificationRepository.savePreferences(updated);
    await _scheduler.rescheduleAll(updated);
    notifyListeners();
  }

  Future<void> markAsRead(String notificationId) async {
    final index = _notifications.indexWhere((n) => n.id == notificationId);
    if (index != -1) {
      _notifications[index] = _notifications[index].copyWith(isRead: true);
      _unreadCount = _notifications.where((n) => !n.isRead).length;
      await _notificationRepository.markAsRead(notificationId);
      notifyListeners();
    }
  }

  Future<void> markAllAsRead() async {
    _notifications = _notifications.map((n) => n.copyWith(isRead: true)).toList();
    _unreadCount = 0;
    await _notificationRepository.markAllAsRead();
    notifyListeners();
  }
}
```

**NotificationScheduler (Local Scheduling):**
```dart
class NotificationScheduler {
  final FlutterLocalNotificationsPlugin _localNotifications;

  static const int dailyReminderId = 1001;
  static const int streakWarningId = 1002;
  static const int reviewDueId = 1003;
  static const int quotaResetId = 1004;

  Future<void> scheduleAll(NotificationPreferences prefs) async {
    await cancelAll();
    if (prefs.dailyReminderEnabled) {
      await _scheduleDailyReminder(prefs.dailyReminderTime);
    }
    if (prefs.streakWarningEnabled) {
      await _scheduleStreakWarning(prefs.streakWarningTime);
    }
    if (prefs.reviewReminderEnabled) {
      await _scheduleReviewReminder(prefs.reviewReminderTime);
    }
    if (prefs.quotaResetEnabled) {
      await _scheduleQuotaReset();
    }
  }

  Future<void> _scheduleDailyReminder(TimeOfDay time) async {
    await _localNotifications.zonedSchedule(
      dailyReminderId,
      'Time to practice!',
      'Your daily English lesson is waiting. Keep your streak alive!',
      _nextInstance(time),
      _channelDetails('daily_reminder', 'Daily Reminders'),
      matchDateTimeComponents: DateTimeComponents.time,
      androidScheduleMode: AndroidScheduleMode.inexactAllowWhileIdle,
    );
  }

  Future<void> _scheduleStreakWarning(TimeOfDay time) async {
    await _localNotifications.zonedSchedule(
      streakWarningId,
      'Don\'t lose your streak! 🔥',
      'You haven\'t practiced today. Complete one lesson to keep going!',
      _nextInstance(time),
      _channelDetails('streak_warning', 'Streak Warnings', importance: Importance.high),
      matchDateTimeComponents: DateTimeComponents.time,
      androidScheduleMode: AndroidScheduleMode.inexactAllowWhileIdle,
    );
  }

  Future<void> _scheduleReviewReminder(TimeOfDay time) async {
    await _localNotifications.zonedSchedule(
      reviewDueId,
      'Flashcards ready for review',
      'You have cards due for review. Quick revision keeps knowledge fresh!',
      _nextInstance(time),
      _channelDetails('review_due', 'Review Reminders'),
      matchDateTimeComponents: DateTimeComponents.time,
      androidScheduleMode: AndroidScheduleMode.inexactAllowWhileIdle,
    );
  }

  Future<void> _scheduleQuotaReset() async {
    await _localNotifications.zonedSchedule(
      quotaResetId,
      'Quota refreshed!',
```

```dart
Future<void> main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await Firebase.initializeApp(options: DefaultFirebaseOptions.currentPlatform);

  // Request notification permission (iOS)
  final messaging = FirebaseMessaging.instance;
  final settings = await messaging.requestPermission(
    alert: true,
    badge: true,
    sound: true,
  );

  // Initialize local notifications
  final localNotifications = FlutterLocalNotificationsPlugin();
  await localNotifications.initialize(
    const InitializationSettings(
      android: AndroidInitializationSettings('@mipmap/ic_launcher'),
      iOS: DarwinInitializationSettings(),
    ),
    onDidReceiveNotificationResponse: (response) {
      // Handle notification tap → deep link navigation
      final deepLink = response.payload;
      if (deepLink != null) {
        GoRouter.of(navigatorKey.currentContext!).push(deepLink);
      }
    },
  );

  // Get FCM token and save to Firestore
  final token = await messaging.getToken();
  // Save token after auth (in AuthProvider.signIn)

  // Handle foreground messages
  FirebaseMessaging.onMessage.listen((RemoteMessage message) {
    // Show local notification while app is open
    localNotifications.show(
      message.hashCode,
      message.notification?.title,
      message.notification?.body,
      const NotificationDetails(
        android: AndroidNotificationDetails(
          'learning_reminders',
          'Learning Reminders',
          importance: Importance.defaultImportance,
        ),
      ),
      payload: message.data['deepLink'],
    );
  });

  // Handle background/terminated tap
  FirebaseMessaging.onMessageOpenedApp.listen((RemoteMessage message) {
    final deepLink = message.data['deepLink'];
    if (deepLink != null) {
      GoRouter.of(navigatorKey.currentContext!).push(deepLink);
    }
  });

  runApp(const AuraCoachApp());
}
```

## 17. Clean Code Principles

All code follows these standards:

1. **Naming Consistency:**
   - Variables: camelCase (userProfile, conversationId)
   - Classes: PascalCase (UserProfile, ConversationModel)
   - Constants: camelCase with `const` keyword (maxTokens, defaultTimeout)
   - Private members: _underscore prefix (_aiRepository, _currentUser)

2. **No Redundant Comments:**
   - Only comment on WHY, not WHAT
   - Bad: `// Increment counter` above `count++`
   - Good: `// Retry on rate-limit (exponential backoff recommended)`

3. **One Responsibility:**
   - Each provider manages one feature's state
   - Each repository handles one data source
   - Each screen displays one major section

4. **DRY (Don't Repeat Yourself):**
   - Shared widgets extracted to `lib/shared/widgets/`
   - Constants in `lib/core/constants/`
   - Reusable utilities in `lib/core/utils/`

5. **Error Handling:**
   - All async operations wrapped in try-catch
   - Error messages friendly (no stack traces in UI)
   - Fallback values for failed API calls

## 18. Cross-Cutting Concerns

### 18.1 Error Handling

All errors bubble up with user-friendly messages:
- Network errors: "No internet connection. Please try again."
- API errors: "Could not generate response. Please try again."
- Auth errors: "Authentication failed. Please sign in again."
- Quota errors: "Daily limit reached. Upgrade to continue."

### 18.2 Loading States

- Global loading: SplashScreen + AuthScreen
- Section loading: Feature screens show spinner in affected section
- Button loading: ClayButton shows spinner + disables input

### 18.3 Offline Behavior

- Read: SharedPreferences (cached conversations, saved items)
- Write: Queue to sync_queue, retry when online
- Sync indicator: Show "Syncing..." badge in affected sections

### 18.4 Conversation Persistence

Every chat mode supports:
- **Auto-save:** After each message, save conversationId locally + Firestore
- **Resume:** Load conversationId from route → Fetch history → Display
- **Archive:** Conversations marked completed → Move to history
