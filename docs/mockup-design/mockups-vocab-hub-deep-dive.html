ephemeral
/ɪˈfɛm.ər.əl/
adjective
Definition
Lasting for a very short time; transitory.
Nghĩa tiếng Việt
Phù du, chóng tàn — tồn tại thời gian ngắn rồi biến mất.
Examples · 3
"The beauty of cherry blossoms is ephemeral — barely two weeks before they fall."
"Fame in viral videos is ephemeral; last week's star is this week's forgotten meme."
"She knew their summer romance would be ephemeral, and that made it all the more precious."

Vocab Hub mode main color = coral (#E8927C)

Nguồn truth: home_screen.dart:457 accentColor: AppColors.coral. Tất cả CTA · chips · badges · accents bên trong Vocab Hub đều dùng coral để khớp với mode card ở Home screen. Icons 5 tool khớp chính xác với mockups-mode-deep-dive.html: 🔍 🧠 🃏 📚 📈.

ElementColor
Primary CTA buttons (Save, Analyze, Practice…)coral — mode main color
Mode identifier pill (top-right app bar)coral
Mode hero gradient (list home)coral → gold warm diagonal
Feature icon tile bg (5 rows)coral @ 14% alpha
Badge pills (Pro · 12 due · 84)coral solid

Titles & body textwarm-dark (unchanged)

Nguyên tắc: Mỗi mode có MỘT màu main color = accentColor của mode card ở Home screen. Vocab Hub = coral, Story Mode = purple, Tone Translator = gold, Scenario Coach = teal. Khi user vào sub-screen trong một mode, mọi primary CTA + icon bg + chip active + badge + progress fill đều dùng mode color đó — để giữ brand moment liền mạch với card Home. Neo-clay pattern (2px warm-dark border + 3×3 warm-dark shadow) là invariant, chỉ đổi màu fill.

Changed vs current TabBar implementation

  • Bỏ TabBar 5-tab — thay bằng vertical card list (deep-dive pattern).
  • Bỏ Describe tab — không có trong spec "5 Powerful tools". Reverse-dictionary có thể rebuild thành shortcut nhỏ trong Word Analysis screen nếu anh cần giữ.
  • Thêm Progress Dashboard — có trong spec nhưng trước đây chưa implement.
  • Mode-consistent CTA — mọi primary button trong Vocab Hub dùng coral + neo-clay warm-dark (coral = mode main color). Story Mode sẽ dùng purple, Tone sẽ dùng gold, Scenario dùng teal — mỗi mode giữ màu của mode card ở Home.
  • 5 route mới — /vocab-hub/word-analysis, /vocab-hub/mind-map, /vocab-hub/flashcards, /vocab-hub/library, /vocab-hub/progress. Mỗi screen có ClayBackButton (icon-btn) trở về Vocab Hub home.
